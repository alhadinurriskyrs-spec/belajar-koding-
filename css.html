Belajar CSS

Pengenalan CSS

CSS (Cascading Style Sheets) digunakan untuk memperindah tampilan HTML. Mengatur warna, font, ukuran, layout, animasi, dan responsivitas web.

Struktur Dasar CSS

selector {
    property: value;
}

Contoh:

body {
    background-color: #f0f0f0;
    font-family: Arial, sans-serif;
}
h1 {
    color: blue;
}

Tips CSS

  • Gunakan class/id untuk styling tertentu
  • Gunakan flexbox/grid untuk layout modern
  • Gunakan warna hex atau nama warna

Quiz CSS

1. Properti untuk warna teks?

  • A. background-color
  • B. color
  • C. font-style

2. Menambahkan class di HTML?

  • A. class="nama"
  • B. id="nama"
  • C. style="nama"
