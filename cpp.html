Belajar C++

Pengenalan C++

C++ adalah bahasa pemrograman berorientasi objek yang digunakan untuk membuat software, game, dan aplikasi sistem. C++ terkenal cepat dan fleksibel.

Struktur Dasar C++

#include <iostream>
using namespace std;

int main() {
    cout << "Halo Dunia!" << endl;
    return 0;
}
  • #include <iostream> → Library untuk input/output
  • using namespace std → Menggunakan standar namespace
  • int main() → Titik masuk program
  • cout → Output teks ke console

Syntax Penting C++

  • Variabel: int, float, double, string, bool
  • Loop: for, while, do-while
  • Kondisi: if, else if, else, switch
  • Fungsi: Deklarasi & definisi fungsi

Contoh Program Sederhana

#include <iostream>
using namespace std;

int main() {
    int umur = 20;
    if(umur >= 18){
        cout << "Dewasa" << endl;
    } else {
        cout << "Belum dewasa" << endl;
    }
    return 0;
}

Tips C++

  • Gunakan komentar untuk dokumentasi
  • Gunakan IDE atau compiler modern
  • Perhatikan tanda titik koma (;) di setiap statement
