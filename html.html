Belajar HTML

Pengenalan HTML

HTML adalah bahasa markup untuk membuat struktur halaman web. Semua website menggunakan HTML sebagai kerangka dasar. HTML bersifat statis dan ditampilkan di browser.

Struktur Dasar HTML

<!DOCTYPE html>
<html>
<head>
    <title>Judul Halaman</title>
</head>
<body>
    Konten halaman
</body>
</html>
  • <!DOCTYPE html> → Menandai HTML5
  • <html> → Tag utama
  • <head> → Metadata
  • <body> → Konten yang tampil

Syntax Penting

  • <h1>-<h6> → Heading
  • <p> → Paragraf
  • <a href=""> → Link
  • <img src=""> → Gambar
  • <ul>/<ol>/<li> → Daftar

Contoh HTML

<!DOCTYPE html>
<html>
<head>
    <title>Contoh</title>
</head>
<body>
    <h1>Halo Dunia</h1>
    <p>Paragraf pertama</p>
    <a href="https://google.com">Google</a>
</body>
</html>

Tips HTML

  • Gunakan <h1> hanya sekali per halaman
  • Validasi HTML di W3C Validator

Quiz HTML

1. Tag link adalah … ?

  • A. <a>
  • B. <link>
  • C. <href>

2. Tag paragraf adalah … ?

  • A. <h1>
  • B. <div>
  • C. <p>
